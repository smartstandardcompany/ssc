# SSC Track - ERP System PRD

## Original Problem Statement
A comprehensive business management ERP system for tracking sales, expenses, supplier payments, HR operations, restaurant POS, stock management, and more.

## Core Modules
- **Financial Management:** Sales, Expenses, Supplier Payments, Customer Credit, ZATCA Invoicing, P&L Reporting
- **HR Management:** Employee DB, Salary Payments, Leave Tracking, Loan Management, Employee Self-Service Portal with Salary History
- **Stock Management:** Inventory, Stock-in/out, Multi-branch transfers, Low-stock alerts
- **Restaurant Operations:** POS, KDS, Order Status Display, Table Management
- **CCTV Security:** Hikvision DVR integration (Local, Remote, Cloud), Live View with snapshot display, AI analytics, Setup Guide
- **Administration:** RBAC, Branding, Dark mode, Multi-language, PWA

## Latest Session Implementations (Mar 2026)
1. **Employee Portal - Salary Record tab:** Month-by-month salary payment history
2. **CCTV Live View Fix:** Actual camera snapshot display via Hikvision ISAPI with auto-refresh
3. **CCTV Remote DVR Support:** 3 connection types (Remote IP, Local IP, Cloud). Removed cloud DVR block. Port forwarding guidance.
4. **CCTV Setup Guide:** Step-by-step port forwarding, DDNS setup, 4 TV display options (HDMI, Web App, Hik-Connect, iVMS-4200)
5. **DVR ID collision fix:** Added microseconds to prevent ID collisions
